Audio Classification

We provide several test files from AudioSet and SPCv2 to briefly illustrate features of these classification tasks.

Speech Command Ver.2

The dataset presents a 35-class single-label speech command recognition task. The decision in test set is made by selecting the command that is predicted to have the highest (top-1) probability. The top-1 accuracy in test set is 97.8% for both LibriTTS and AudioSet pretraining. Eight segments are shared in this demo, whose ground truth labels are listed on top of the audio file. The model correctly predicted the command for all eight files.

AudioSet

AudioSet presents a 527-class multi-label tasks, where various sound sources, events and scenarios are included. Models are finetuned in AudioSet with two settings: AudioSet-2M (summation of imbalance and balanced subsets) and AudioSet-20k (balance subset alone). The common metric in AudioSet is mean average precision, which does not require a 0/1 binary decision (0-100% probability is required instead). For better understanding to the task, we use thresholds optimized on test set itself to produce multi-label decisions.

Label classes are presented on top of the audio file, while predicted tags from AudioSet-2M and AudioSet-20K are presented below the audio file.
